## Setup

1. Clone the repository

```
git clone https://github.com/Ankitkr5858/chatgpt-clone.git
```

2. Install client dependencies

```
cd ChatGPT
npm install
```

3. Get your Open AI API key from [Open AI API Key](https://platform.openai.com/account/api-keys)

## Usage

1. Start the app

```
npm run dev
```
